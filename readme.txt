Перед запуском stt_app необходимо определить параметры в файле конфигурации и положить его рядом с stt_app.

Доступны следующие параметры:

audio_path - путь до аудио файла,
result_path - путь до файла с результатами (wav для изменения аудио или JSON для перевода речи в текст),
function - achange - изменение скорости/громкости, transcript - перевод речи в текст,
speed - скорость нового файла,
volume - громкость нового файла,
model_lang - язык модели для перевода (ru/en),
log_dir - папка, куда будут записываться логи процесса (этапы, ошибки).


Особых требований к системе нет. Linux.
Модели небольшие, памяти много не требуется.
Файл stt_app собирался в google colaboratory в среде без видеокарты. Python 3.10.
